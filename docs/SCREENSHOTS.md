# Screenshot Requirements

Each screenshot should be 1920x1080 PNG captured in-game.

## 1. Main Menu
- Shows the globe rotating
- All menu buttons visible
- Title "NORAD WAR SIMULATOR" prominent
- Version number visible

## 2. Missile Tracking
- Globe view with missile in flight
- Trajectory line visible
- DEFCON indicator showing threat level
- HUD with interceptor counts

## 3. Interceptor Launch
- Player clicking to launch interceptor
- Intercept trajectory visible
- Missile and interceptor paths converging
- Success probability displayed

## 4. Campaign Tech Tree
- Tech tree showing available upgrades
- GBI, THAAD, Patriot branches
- Tech points visible
- Locked vs unlocked tech

## 5. Scenario Editor
- Wave editor visible
- Launch site selection
- Target city selection
- Scenario preview

## 6. Multiplayer Lobby
- Player list with ready status
- Game mode selection
- Chat visible
- Start button

## 7. Achievement Screen
- Achievement grid visible
- Progress bar
- Some achievements unlocked, some locked
- Stats display

## 8. DEFCON Alert
- DEFCON warning popup
- Alert klaxon visual effect
- Missile swarm incoming
- Critical alert styling

## 9. Nuclear Detonation
- Explosion effect on globe
- Impact marker
- Damage radius visualization
- City highlighted

## 10. Satellite Detection
- Satellite network visible
- Launch detection indicator
- Early warning display
- Globe view with satellite orbits

## How to Capture

1. Open game in Godot editor
2. Press F5 to run
3. Navigate to each screen
4. Press F12 or use screenshot tool
5. Save as screenshot_01.png, etc.
